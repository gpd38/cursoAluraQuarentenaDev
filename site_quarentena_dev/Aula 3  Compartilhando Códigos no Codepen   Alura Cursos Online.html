Respostas dos exercícios da aula2

O primeiro exercicio pedia para calcular o IMC ou índice de massa corporal. A fórmula é peso dividido pela altura ao quadrado, então poderiamos ter feito o seguinte

meupeso = prompt('qual seu peso?')
minhaaltura = prompt('qual sua altura')

document.write('o meu imc é ' + (meupeso / (minhaaltura*minhaaltura)))

Lembrando de usar números com ponto ao invés de virgula no momento de preencher os dados.

O segundo exercício pedia a média de idades da familia do Guilherme:

guilherme = parseInt(prompt('A idade do Guilherme é: '))
carol = parseInt(prompt('A idade da Carol é: '))
marcia = parseInt(prompt('A idade da Márcia é: '))
patricia = parseInt(prompt('A idade da Patricia é: '))
soma = ((guilherme + carol + marcia + patricia)/4)

document.write('A media de idade da familia do Guilherme é: ' + soma + ' anos')

Nesse exercício tinha uma pegadinha, precisavamos converter a idade em inteiro, pois o comando prompt() retorna um texto para gente. Para fazer isso poderiamos ter utilizado o comando parseInt antes do prompt.

Estavamos fazendo operações com texto por causa do prompt, então o sinal de + o JavaScript entende como se estivessemos falando junte esse número com esse outro número e não some os dois números, por isso talvez você não tenha conseguido fazer a soma.

Não se sinta mal caso não tenha conseguido resolver esse exercício, estamos aprendendo junto e lidar com erros e pesquisar soluções fazem parte da nossa vida de programadores e programadoras.
